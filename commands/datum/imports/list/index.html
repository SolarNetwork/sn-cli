    Datum Imports List¶

    Show datum import job status matching a search filter.

    Usage¶

    s10k datum imports list
    [-state=jobState[,jobState...]]...
    [-mode=<displayMode>]

    Options¶

    Option Long Version Description
    -state= --job-state= the job state(s) to match, one of Staged, Retracted, Queued, Claimed, Executing, or Completed; see the wiki for more info
    -mode= --display-mode= the format to display the data as, one of CSV, JSON, or PRETTY; defaults to PRETTY

    Output¶

    A listing of all matching job states.

    Examples¶
